# Patient-Readmission Analysis

This project analyzes hospital patient readmission data to identify key trends and insights that can help healthcare providers reduce readmission rates and improve patient care outcomes.

# Repository Contents

 File Description 
Patient_Readmission - Raw Data.csv**  The original dataset containing patient admission and readmission details. 
SQL Folder containing SQL scripts used for data cleaning, transformation, and analysis. 
SQL OUTPUT.csv Processed dataset or query results generated from the SQL scripts. 
Patient Readmission Dashboard.pdf Visualization dashboard summarizing key findings and metrics. 
Patient Readmission Presentation.pptx PowerPoint presentation summarizing the overall analysis and recommendations. 
MIRO PLANNING (2).pdf Project planning and workflow document outlining tasks and milestones. 

# Project Objective

The main goal of this project is to:
- Analyze patient data to understand patterns in hospital readmissions.
- Identify key factors contributing to frequent readmissions.
- Provide actionable insights to reduce readmission rates and improve healthcare efficiency.

#Tools & Technologies Used

- SQL – Data cleaning and transformation  
- Google Looker Studio – Data visualization and dashboard creation  
- Microsoft PowerPoint – Presentation of findings  
- Miro – Project planning and collaboration  


# Key Insights (from Dashboard & Analysis)

- Visual summaries show patient demographics, admission types, and readmission frequencies.  
- Identified trends related to "length of stay", "number of diagnoses", and "previous admissions".  
- Recommendations provided to improve patient follow-up and optimize resource allocation.
